If Something Fails — Checklist

1
ไปที่ supabase.com → เปิด Project → ถ้าขึ้น "Project is paused" ให้กด Restore/Unpause
2
ไปที่ Settings → API → copy URL และ anon key ใหม่ → แล้วแจ้งมาเพื่ออัปเดต config.js
3
ถ้า Step 3 fail → กลับไป Sign In อีกครั้ง session อาจหมดอายุ
4
ถ้า Step 4–5 fail → ต้องรัน SQL ใน Supabase SQL Editor เพื่อเปิด RLS permissions ให้กับ table
5
ตรวจว่า XAMPP ยังเปิดอยู่: Apache ต้อง status Running
